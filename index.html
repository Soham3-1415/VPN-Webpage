About



I made this project for a few reasons:
  • - learn to create APIs
  • - work with ReCaptcha
  • - use Hashicorp's Vault to run a PKI
  • - use Node.js
  • - practice writing secure code
  • - scale an OpenVPN instance

As such, this is a personal project of mine and should not be taken as a subsitute for an actually reliable VPN. In fact, don't count on this at all. This application has not been vetted.

Don't expect security.
Don't expect privacy.
Don't expect reliability.
Don't abuse this service.

In addition, consider if you really need a VPN. Companies often try to sell VPNs using scare tactics. I'm not doing a writeup here, but keep in mind what a VPN actually does. It's also quite easy and cheap to run your own VPN.


To connect, you must have a certificate. Currently, certificates are issued for 7 days.
I use OpenVPN. You can connect with UDP, TCP, or with TCP wrapped with TLS.
I try to not maintain connection logs, but no promises are being made.
Connections must be made using AES-256-GCM. TLS 1.2 support is required.

Port 1194 is for UDP and TCP connections.
TLS wrapped connections must be made to port 80.
